OVERVIEW

DUportal is a free Web portal and online community. Backed end by Access database, DUportal contains numerous advanced features such as Web-based administration, Articles, Banner Ads, Event Calendar, Classified Ads, Web link directory, Downloads, Message Board, Picture Gallery, News, E-Commerce, Polls and Business Directory, and more which can be downloaded online. All modules are customizable via Web-based Admin panel, together with size, skins and themes. DUportal is an excellent solution for your Web portal and online community needs. Start building a community online by using this free Web application.

SPECIFICATIONS

DUportal is an easy-to-use ASP-based Web Portal application which can be completely customized via Web browsers:

- Utilizing Microsoft Active Server Pages that can be tested and hosted on any Windows-based PCs and servers
- Unlimited modules and categories
- Unlimited entries with images and full description
- Fast, real time image uploading
- Complete search feature
- Rating system included
- E-Commerce with Paypal Shopping Cart and multiple currency.

DUportal comes with a Web-based Admin panel, you will be able to perform all of the following:

- Password-protected Admin panel
- Portal Configuration
- Include/exclude the modules of your portal
- Users Manager
- Add new category, delete existing categories
- Approve new entries
- Add/edit/delete entries

Add-ons and updates can be found online

Please check the Professional version of this application for more advance features.

SET UP

After you unzipped DUportal.zip, use an FTP program such as CuteFTP to upload all files and folders to the root of your site.

Log into Admin panel at:

http://www.yourdomain.com/admin

The default Admin account is

Admin ID: admin
Password: password

You will be able to change your password using the Users Manager

- Click on Configuration link on the menu to set up your new portal.

- Click on Channels link to customize the modules of your portal

- Click on Categories to add/edit/delete the categories of each module

- After you have created polls, you will need to click on Polls link to activate the new poll.
